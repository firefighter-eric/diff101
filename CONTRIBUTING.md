# Contributing

Thanks for considering a contribution!

## Development setup (uv)

```bash
uv venv --python 3.12
uv sync --python 3.12
uv pip install "torch"
```

## Run the app

```bash
uv run --python 3.12 python app.py
```

## Reporting issues

Please include:
- OS + Python version
- GPU/CPU details
- Full error logs
- Steps to reproduce

## Pull requests

- Keep changes focused
- Update README if behavior changes
- Include testing steps or a minimal manual verification note
